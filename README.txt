Store Rating System - Final Project Submission

This project is a full-stack store rating system with three distinct roles: 
Admin, Store Owner, and Normal User.

--- HOW TO SETUP ---

1. DATABASE:
   - Import the 'store_rating_system.sql' file (found in this folder) into your 
     local MySQL/phpMyAdmin. 
   - Database name: store_rating_system

2. BACKEND:
   - Go to the /backend folder.
   - Run 'npm install'.
   - Ensure you have a .env file with your local DB credentials.
   - Run 'node server.js' (Starts on port 5000).

3. FRONTEND:
   - Go to the /frontend folder.
   - Run 'npm install'.
   - Run 'npm start' (Starts on port 3000).
